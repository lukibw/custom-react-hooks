# Custom React Hooks

Simple, but very useful react custom hooks:

- usePreviousState
- useFetch
- useLocalStorage
- useOnline
- useWindowEventListener
- useClickOutside
- useScrollPosition
- useMousePosition
- useWindowDimensions
- useTimeout
- useInterval
- useDebouncedValue
- useDebouncedFunction
- useThrottledFunction

## Install

Inside your React project, run:

`yarn add custom-react-hooks`

or with npm:

`npm install custom-react-hooks`
